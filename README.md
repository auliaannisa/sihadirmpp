\# SIHADIR MPP



Sistem Informasi Kehadiran (Absensi) berbasis Web menggunakan PHP dan MySQL.



\## Fitur

\- Login admin

\- Absensi dengan upload selfie

\- Upload surat izin

\- Rekap absensi

\- Export ke Excel



\## Teknologi

\- PHP Native

\- MySQL

\- HTML, CSS



\## Cara Install (Local)

1\. Clone repository

2\. Pindahkan ke folder htdocs

3\. Import database ke MySQL

4\. Atur koneksi database di config.php

5\. Jalankan melalui browser



\## Catatan

Folder uploads harus dibuat manual karena tidak ikut repository.
